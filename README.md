# Hacknet存档编辑器

![Python 3.7](https://img.shields.io/badge/Python-3.7-blue.svg)
![GPLv3 License](https://img.shields.io/badge/License-GPLv3-brightgreen.svg)

基于Python的Hacknet游戏存档编辑工具，支持对游戏存档文件的读写操作，提供全面的节点编辑功能、文件系统管理、DEC文件加解密、内存转储分析以及自定义节点创建等特性。

## ✨ 功能特性

### 基础功能
1. **节点信息编辑**
   - 修改节点IP地址和显示名称
   - 编辑管理员密码和防火墙密码
   - 自定义开放端口列表
   - 调整追踪时间、代理时间和破解难度
   - 实时查看节点解锁状态

2. **可视化操作界面**
   - 使用Tkinter构建的交互式表格界面
   - 右键菜单提供编辑、新建节点、删除节点操作
   - 实时显示当前存档玩家名称
   - 自动刷新修改后的存档数据

3. **文件管理器**
   - 双击节点打开该计算机的完整文件系统
   - 右键菜单支持**新建、重命名、复制、粘贴、删除**文件或文件夹
   - 顶部地址栏实时显示当前位置，支持输入路径快速跳转
   - 跨窗口剪切板，复制内容可在不同计算机之间粘贴
   - 粘贴同名文件时弹出对比窗口，可选择覆盖或跳过
   - 编辑文本时自动转换 Hacknet 标记（`|##LAB##|` 等）与普通字符，确保存档格式不被特殊符号污染

4. **自定义节点创建**
   - 右键菜单一键创建新节点，支持设置 IP、名称、类型、坐标、管理员密码
   - 支持 **15+ 种守护进程**（WebServer、MailServer、MedicalDatabase、DeathRowDatabase、PointClicker、MissionHubServer 等）
   - 创建节点时自动初始化对应守护进程所需的文件夹和初始数据文件
   - 高级安全设置：自定义开放端口、骇入端口数、防火墙开关/密码、追踪时间、代理时间

5. **DEC 文件加解密器**
   - 独立加解密窗口，支持明文 ↔ 密文互转
   - 左右文本框等高等宽，共用密码输入，解密自动回填标题和 IP
   - 菜单支持打开/保存 `.txt` 和 `.dec` 文件

6. **DEC 文件解密器**
   - 自动收集全存档中所有 `.dec` 加密文件
   - 双击弹出密码窗口，支持指定密码解密或暴力破解
   - 递归解密多层嵌套，解密结果可保存至玩家 `/home` 或导出本地

7. **内存转储（Memory）分析**
   - 文件管理器内显示节点的 `Memory` 数据（紧凑格式，与游戏内一致）
   - 内存转储查看器：列出所有包含内存转储的计算机，双击可查看详情
   - 支持保存内存转储为标准 `.mem` 文件至 `/home/MemDumps`
   - 智能解密标准内存转储文件（`.mem`），并自动识别内嵌 DEC 加密内容

### 特殊功能
1. **一键全程序获取**
   - 自动补全玩家 `/bin` 目录下所有缺失的 `.exe` 文件
   - 包含游戏内未启用的废案程序（`confloodEOS.exe`、`GitTunnel.exe`、`WoWHack.exe` 等）

2. **管理员权限获取**
   - 一键获得所有节点管理员权限
   - 一键获得邮件服务器（Jmail/boatmail）的所有账号

3. **安全强化**
   - 无懈可击模式：开启EnTech防御、瞬间追踪、解锁全端口、随机生成12位复杂防火墙密码

4. **进度解锁**
   - 一键解锁全部节点访问权限
   - 批量管理节点解锁状态

5. **菜单结构优化**
   - "功能"菜单：DEC文件查看器、内存转储查看器、DEC文件加解密器
   - "特殊"菜单：一键解锁全节点、一键管理员权限、坚不可摧、一键获得可执行文件

## 🚀 快速开始

### 环境要求
- Python 3.7.0+
- 依赖库：`tkinter`, `xml.etree.ElementTree`（Python标准库，无需额外安装）

### 使用步骤
1. 下载项目：
```bash
git clone https://github.com/TesterNaN/Hacknet_Saver.git
cd Hacknet_Saver
```
2. 运行程序：
```bash
python main.py
```

3. 操作流程：
   - 文件 → 打开存档 → 选择存档文件
   - 双击节点打开文件管理器，右键节点编辑/新建/删除节点，右键文件/文件夹进行操作
   - 使用"功能"与"特殊"菜单执行各类操作
   - 文件 → 覆盖保存/另存为 完成修改
