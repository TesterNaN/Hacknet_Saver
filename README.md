# Hacknet存档编辑器
![Python 3.7](https://img.shields.io/badge/Python-3.7-blue.svg)
![GPLv3 License](https://img.shields.io/badge/License-GPLv3-brightgreen.svg)

基于Python的Hacknet游戏存档编辑工具，支持对游戏存档文件的读写操作，提供全面的节点编辑功能和游戏辅助特性。

## ✨ 功能特性

### 基础功能
1. **节点信息编辑**
   - 修改节点IP地址和显示名称
   - 编辑管理员密码和防火墙密码
   - 自定义开放端口列表
   - 调整追踪时间和破解难度
   - 实时查看节点解锁状态

2. **可视化操作界面**
   - 使用Tkinter构建的交互式表格界面
   - 双击单元格直接修改数值
   - 实时显示当前存档玩家名称
   - 自动刷新修改后的存档数据

### 特殊功能
1. **管理员权限获取**
   - 一键获得所有节点管理员权限
   - 一键获得邮件服务器（Jmail/boatmail）的所有账号

2. **安全强化**
   - 无懈可击模式：
     - 开启EnTech防御模式
     - 瞬间追踪
     - 解锁全端口（80/25/21/22/1433/3659/104）
     - 随机生成12位复杂防火墙密码

3. **进度解锁**
   - 一键解锁全部节点访问权限
   - 批量管理节点解锁状态

## 🚀 快速开始

### 环境要求
- Python 3.7.0+
- 依赖库：`tkinter`, `xml.etree.ElementTree`（Python标准库，无需额外安装）

### 使用步骤
1. 下载项目：
```bash
git clone https://github.com/TesterNaN/Hacknet_Saver.git
cd Hacknet_Saver
```
2. 运行程序：
```bash
python main.py
```

3. 操作流程：
   - 文件 → 打开存档 → 选择存档文件
   - 双击表格单元格直接编辑
   - 使用功能菜单执行特殊操作
   - 文件 → 覆盖保存/另存为 完成修改
